Interface to Hierarchical Agents

1

execute_plan(plan) -> RunResult

Main entry point. Takes a plan from upper agent, returns execution outcome.

2

RecoveryAdvisor hook

Optional callback for upper agent to provide recovery hints.

3

RunObserver / event stream

Real-time event stream for monitoring: telemetry, decisions, errors.

Design Philosophy

  • I do not touch planning
  • I only provide reliable execution backend
  • Alan agents only need these three interfaces
