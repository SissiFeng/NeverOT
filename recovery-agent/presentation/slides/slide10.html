Safety and Irreversibility Guarantees

Core Guarantees

1

Unsafe Preemption

Safety violations trigger immediate cool_down before any other action

2

Irreversible Action Tracking

Sample status tracked: intact / compromised / destroyed

3

Terminal Invariants on Abort

Shutdown verifies safe state: heating == False, status == idle

Examples

temp > 130C

Immediate cool_down, then degrade or abort

electrochem_started

Sample marked compromised, cannot revert

Any agent built on this layer will not cause equipment damage.
