  步骤 2: 配置环境变量

  创建 .env 文件：

  # ========================================
  # 真实硬件配置
  # ========================================

  # 切换到真实硬件模式
  ADAPTER_MODE=battery_lab

  # 禁用 dry_run（真正执行）
  ADAPTER_DRY_RUN=false

  # OT-2 机器人
  ROBOT_IP=192.168.1.100  # ← 替换为你的实际 IP

  # 电化学工作站
  SQUIDSTAT_PORT=/dev/ttyUSB0  # macOS/Linux
  # SQUIDSTAT_PORT=COM3        # Windows

  # USB 继电器
  RELAY_PORT=COM11

  # ========================================
  # 或者保持安全模式测试（推荐第一次）
  # ========================================

  # 硬件连接但不真正执行
  ADAPTER_MODE=battery_lab
  ADAPTER_DRY_RUN=true  # ← 验证通信，不执行动作
